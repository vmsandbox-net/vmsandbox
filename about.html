Debian GNU/Linux

Various authors and copyright holders — Multiple open source licenses

The root filesystem used by each virtual machine in vmsandbox is built from Debian GNU/Linux packages, cross-compiled for the RISC-V 64-bit architecture using debootstrap. Debian is one of the oldest and most widely used Linux distributions, maintained entirely by volunteer contributors according to the Debian Social Contract and the Debian Free Software Guidelines.

The userspace environment inside each VM includes the GNU C Library (glibc), the GNU core utilities (coreutils), bash, iproute2, tcpdump, and the other networking tools used in the lab exercises. Debian packages installed on the filesystem include PostgreSQL, etcd, bird, irssi, and the other lab-specific software described below.

Each Debian package carries its own copyright and licence, documented in /usr/share/doc/<package>/copyright inside the VM filesystem. The Debian project maintains a comprehensive licence overview at debian.org/legal/licenses.

Debian's commitment to free software, rigorous package quality standards, and stable release cycle make it a natural choice as the foundation for an educational environment where users are expected to read, understand, and modify the software they are running.
